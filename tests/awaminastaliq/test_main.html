Awami Nastaliq - Webfont Test

This is a test of the font family based on three sources: Google Fonts (GF), fonts.languagetechnology.org (FLO), and local path references (LPR). It must be viewed on a machine that does not have the font locally installed or it may not actually use the Google Font.

This font is not available on Google Fonts. We have removed lines referencing Google Fonts. If it is ever onboarded, we can add those lines.
